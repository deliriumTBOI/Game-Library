<h1>Game Library</h1>
<p><b>Game Library</b> — это REST API для управления игровой библиотекой</p>
<h2>Функционал</h2>
<o><li>Поиск игры по названию</li>
   <li>Поиск игры по информации о компании</li></o>
<h2>Использованные технологии</h2>
<o><li>Java 17</li>
   <li>Maven</li>
   <li>Spring Boot</li>
   <li>Spring Web</li>
   <li>REST API</li></o>
